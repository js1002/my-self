//==在进行比较的时候，对于不同的数据类型之间转换的规律如下： //1) 对象==对象 // 比较的是引用的地址，一般情况下都不相等，例如：{}=={} ->false []==[] ->false //2) 对象==字符串 // 先把对象转换为字符串(调用toString方法)，然后在进行比较，例如：[]=="" ->true //3) 对象==布尔 // 两边都转换为数字，对象首先转换为字符串，然后在把字符串转换为数字(调用Number进行严格转换 Number("")->0)；布尔类型转换为数字，true->1 false->0；例如：[]==false ->true //4) 对象==数字 // 对象转化为数字，例如：[]==1 ->false //5) 数字==布尔 // 布尔转化为数字，例如：true==2 ->false //6) 字符串==布尔 // 都转换为数字，例如：""==false ->true //7) 字符串==数字 // 字符串转换为数字，例如："35px"==35 ->false //8) null==undefined ->true // null、undefined和其他的任何数据类型值都不相等 //9) NaN==NaN ->false // NaN和任何的数据类型值都不相等
